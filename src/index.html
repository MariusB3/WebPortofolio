Smart Bin

"Smart Bin" este proiectul meu de licență în curs de dezvoltare. Acesta implică implementarea unui sistem care utilizează recunoașterea obiectelor pentru sortarea automată a gunoiului. Prin intermediul unei camere web, obiectele sunt identificate (de exemplu, o sticlă de plastic) și urmează să fie aruncate în coșul corespunzător folosind două motoare (unul DC și unul servo). Recunoașterea obiectelor este realizată în Python, iar controlul motoarelor este asigurat de un Arduino R4. Comunicarea între Python și Arduino se realizează prin intermediul Arduino Cloud, facilitând integrarea și dezvoltarea ulterioară a sistemului.
